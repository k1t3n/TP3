Spécifications

Vérification nom du client existant.

	Compare si le client est égale a null dans la méthode attribuerCommandes

Vérification plat existant. 

	Compare si le plat est égale a null dans la méthode attribuerCommandes

Vérification format entré client.

	Vérifie si l'entrée ne contient que des lettres dans la méthode createClients

Vérification format entré plat.

	Vérifie si l'entrée ne contient que des lettres, des chiffres et un signe de dollars dans la méthode createPlats

Vérification format entré commande.
	
	Vérifie si le nom est existant, si le plat est existant, si la quantité est un chiffre/nombre et si la facture a le format demandé

Vérification des chiffres erronés.
	
	Vérifie si la quantité de la commande est un chiffre ou un nombre

Facture sur le terminal ET modifier le fichier (Facture-du-(date-heure).txt)
	
	Détermine le nom du fichier avec la classe Calendar, redirection de la console dans un outputStream différent pour save dans un fichier texte apres la premiere execution
